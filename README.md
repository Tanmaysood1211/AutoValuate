# 🚗 AutoValuate

Professional AI-Powered Used Car Valuation Platform built using HTML, CSS, JavaScript, Supabase, and SQL.

## ✨ Features

- 🔐 Secure Login System
- 🚘 Vehicle Registration Lookup
- 📅 Registration Year Detection
- 💰 Original Ex-Showroom Price Estimation
- 🛠️ Condition-Based Vehicle Inspection
- 🤖 AI Valuation Engine
- 📊 Professional Valuation Report
- 👨‍💼 Admin Dashboard
- ☁️ Supabase Database Integration

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Supabase
- SQL
